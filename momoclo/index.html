# ももクロ

2012年。バカソウルで使われていた曲がももクロの「行くぜっ！怪盗少女」だと知り、そこからYoutubeを見てハマってしまった。
その後、運よく取れたももクリ2012のライブビューイングに行き、モノノフ人生が始まる。

詳しくはこちらに長々と書いております。
